YourAppName 是一个基于 NextJS 的 AI SaaS 开发框架，内置丰富的组件和 AI SDK，帮助你快速完成 AI SaaS 项目开发。

## 环境依赖

为确保你能顺利在本地运行和二次开发 YourAppName，请在你的电脑上安装必要的环境。

以下为 Mac 系统开发的环境配置建议：

1. 使用 [fnm](https://github.com/Schniz/fnm) 进行 NodeJS 版本管理

推荐使用以下版本的 NodeJS 和 npm：

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. 使用 [pnpm](https://pnpm.io/installation) 进行包管理

全局安装 pnpm：

```bash
$ npm install -g pnpm
```

推荐使用以下版本的 pnpm：

```bash
$ pnpm -v
9.15.0
```

3. 安装 git 并配置 [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh)，确保能访问你的 Github 仓库

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> 其他操作系统，请自行搜索对应的环境配置方法。运行 YourAppName 代码之前，请确保你的电脑环境已经安装了 nodejs 和 pnpm。
