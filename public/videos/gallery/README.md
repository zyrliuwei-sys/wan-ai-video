## Video Specifications

- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 (16:9)
- **Duration**: 5-10 seconds
- **Frame Rate**: 24fps or 30fps
- **Bitrate**: 2-5 Mbps

## Tips

1. Keep videos short (5-10s) for fast loading
2. Use consistent resolution (1920x1080)
3. Optimize for web (H.264 codec)
4. Consider using placeholders initially (solid color images)
